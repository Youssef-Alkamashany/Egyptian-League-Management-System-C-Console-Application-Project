### 🚀 Key Technical Features
* **🔗 Linked List Architecture:** Custom `TeamNode` and `PlayerNode` for dynamic data storage.
* **📈 Advanced Sorting:** Bubble Sort implementation for Salary and Average Age analytics.
* **🗑️ Cascading Deletion:** Automated memory cleanup to ensure data integrity.
