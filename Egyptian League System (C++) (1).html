Egyptian League System

(C++ with Linked Lists)

Managing Teams and Players using Singly Linked Lists

Slide 1 / 9

Main Idea & Features

Main Idea

  • Console-based system to manage football league data (teams and players).[web:41]
  • Uses singly linked lists instead of static arrays for dynamic storage.[web:39]
  • Each team is stored in a TeamNode, each player in a PlayerNode.

Key Features

  • Add / remove / display teams and players.[web:41]
  • Search team by ID, search player by name.
  • Update a player's team (transfer between clubs).
  • Sort players of a team by salary (descending order).[web:40]
  • Order teams by average age of their players.

Data Structures

TeamNode (Team node)

struct TeamNode { int id; // Team ID string name; // Team name string president; // Club president TeamNode* next; // Pointer to next team };

PlayerNode (Player node)

struct PlayerNode { int id; // Player ID string name; // Player name string teamName; // Team name string position; // Position (GK, RW, CM, ...) int age; // Age double salary; // Salary PlayerNode* next; // Pointer to next player };
Each node stores the data plus a pointer to the next node, which is the classic structure of a singly linked list in C++.[web:39]

LeagueSystem Class

Class Design

class LeagueSystem { private: TeamNode* teamHead; // Head of teams list PlayerNode* playerHead; // Head of players list public: LeagueSystem() { teamHead = NULL; playerHead = NULL; } // Team operations void addTeam(int, string, string); void displayTeams(); void searchTeamByID(int); void removeTeam(int); // Player operations void addPlayer(int, string, string, string, int, double); void displayPlayers(); void searchPlayerByName(string); void removePlayer(int); void updatePlayerTeam(string, string); // Bonus operations void displayPlayersBySalary(string); double calculateAvgAge(string); void displayTeamsByAvgAge(); };
teamHead points to the first team, playerHead points to the first player; all operations traverse from these heads through the linked lists.[web:1][web:41]

Team Operations

Add Team – addTeam()

void addTeam(int id, string name, string president) { TeamNode* newTeam = new TeamNode(); newTeam->id = id; newTeam->name = name; newTeam->president = president; newTeam->next = NULL; if (teamHead == NULL) { // first team teamHead = newTeam; } else { TeamNode* temp = teamHead; while (temp->next != NULL) temp = temp->next; // go to last node temp->next = newTeam; // link at the end } }

Display & Search Teams

  • displayTeams(): traverses from teamHead to NULL and prints id, name, president of each team.[web:41]
  • searchTeamByID(int id): scans nodes until it finds a matching id, otherwise prints "Team not found".
Slide 5 / 9

Player Operations

Add Player – addPlayer()

void addPlayer(int id, string name, string tName, string pos, int age, double sal) { PlayerNode* newPlayer = new PlayerNode(); newPlayer->id = id; newPlayer->name = name; newPlayer->teamName = tName; newPlayer->position = pos; newPlayer->age = age; newPlayer->salary = sal; newPlayer->next = NULL; if (playerHead == NULL) playerHead = newPlayer; // first player else { PlayerNode* temp = playerHead; while (temp->next != NULL) temp = temp->next; // go to last player temp->next = newPlayer; // link at the end } }

Remove Player – removePlayer(int id)

  • If the player is at the head: move playerHead to playerHead->next and delete the first node.[web:41]
  • If the player is in the middle or end: find the previous node, then set temp->next = temp->next->next; and delete the removed node.

Display Players

displayPlayers() prints ID, name, team, position, age, and salary for every player in the list.

Search & Update Players

Search by Name – searchPlayerByName()

Traverses the players list from playerHead and compares temp->name == name. Prints all players with that name and their teams, or a message if no player is found.[web:41]

Update Player Team – updatePlayerTeam()

void updatePlayerTeam(string playerName, string newTeamName) { PlayerNode* temp = playerHead; while (temp != NULL) { if (temp->name == playerName) { temp->teamName = newTeamName; // transfer player to new team return; } temp = temp->next; } }
This function performs a linear search on the players list and then updates the teamName field of the found node.[web:41]

Bonus: Salary & Age

Sort Players by Salary – displayPlayersBySalary()

Uses a double loop over the players linked list (similar to Bubble Sort):
  • Two pointers i and j traverse the list.
  • If i->salary < j->salary, it swaps all data fields between the two nodes (id, name, teamName, position, age, salary).[web:40]
  • After sorting, it prints only players whose teamName matches the selected team.

Average Age – calculateAvgAge()

double calculateAvgAge(string tName) { PlayerNode* temp = playerHead; double sum = 0; int count = 0; while (temp != NULL) { if (temp->teamName == tName) { sum += temp->age; count++; } temp = temp->next; } if (count == 0) return 0; return sum / count; // average age of this team }

Teams by Average Age – displayTeamsByAvgAge()

Uses nested loops over the teams list and calculateAvgAge to swap team data so that teams are ordered by descending average age of their players.
Slide 8 / 9

removeTeam & main()

Remove Team – removeTeam(int id)

  • Searches for the team by id and stores its name (teamName).
  • Deletes that team node from the teams list (handles head and non-head cases).
  • Then repeatedly scans the players list and removes every player whose teamName equals the deleted team.
  • This guarantees no players remain belonging to a removed team.

main() and Menu

LeagueSystem sys; // initial sample data sys.addTeam(101, "Ahly", "Elkhatib"); sys.addTeam(102, "Zamalek", "Labib"); // do-while loop with menu choices 1..11 and 0 to exit
  • The menu lets the user call all operations: add, display, search, update, delete, sorting, and statistics.[web:16]
  • The loop continues until the user enters 0 (Exit).
